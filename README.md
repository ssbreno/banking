# Technologys

- Docker
- NestJS
- NodeJs
- JestJs
- TypeORM
- Postgres
- Husky
- Libs : Prettier, Lint, Swagger

# Requirements

- Docker
- OS Preference Linux or Windows WSL Mode

# Run Project

- Create .env from .env-local
- Run yarn
- docker-compose build
- docker-compose up -d
- Run npm run migrate:run *to create migrations
- Run yarn run start:dev

# Testing

- Only run npm run test
